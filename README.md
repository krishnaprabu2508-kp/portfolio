# Mechanical CAD Portfolio – Krishnaprabu P

This repository contains my **Mechanical Design and CAD Portfolio Website**.
The website showcases my **2D drafting and 3D modeling projects** created using **AutoCAD and SolidWorks**.

The portfolio is built using **HTML, CSS, and JavaScript** and is hosted using **GitHub Pages**.

## 🚀 Deployment

This project is deployed using **GitHub Pages**.

Steps used for deployment:

1. Upload project files to GitHub repository
2. Rename main page as `index.html`
3. Enable GitHub Pages in repository settings
4. Deploy from the `main` branch

GitHub automatically hosts the website.
